## 删除一个虚拟环境

```bash
conda remove --name <虚拟环境名> --all
```

## 如何打包一个 python 项目

- 首先，编写 `MANIFEST.IN` 以及 `pyproject.toml` 
- 在 `base` 环境下执行一下命令：

```bash
rm -f ./dist/*
python3 -m build
python3 -m twine upload ./dist/*
```
